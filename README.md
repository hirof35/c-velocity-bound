# Velocity Simulation (DxLib)
<img width="747" height="778" alt="スクリーンショット 2026-06-11 071551" src="https://github.com/user-attachments/assets/70fc8a19-6fe5-420e-99eb-e574ba275805" />

C言語とDxLib（DXライブラリ）を用いて作成した、重力加速度とバウンド処理（放物運動）の挙動を確認するためのシミュレーションプログラムです。
HTML5 Canvas/JavaScriptで記述された物理演算ロジックをベースに、C言語への移植と60fps環境への最適化（ダブルバッファリング、フレームレート同期、めり込み補正）を行っています。

## 機能・特徴

* **等速直線運動（X軸）**: 画面右端に到達すると、スムーズに左端から再登場します。
* **重力加速度シミュレーション（Y軸）**: 時間経過（フレーム更新）ごとに下方向への速度が加算されるリアルな自由落下を再現しています。
* **完全なバウンド処理**: 画面下部を「床」として判定し、衝突した瞬間に座標を補正して上方向への初速を再与えることで、画面外へのめり込みを防ぎます。
* **60fps最適化**: `ScreenFlip()` を利用したダブルバッファリングにより、チラつきのない滑らかな描画を実現しています。

## 動作環境

* **開発言語**: C言語
* **グラフィックライブラリ**: [DXライブラリ (DxLib)](https://dxlib.xsrv.jp/)
* **対応OS**: Windows 7 / 8 / 10 / 11
* **開発環境例**: Visual Studio 2022 / GCC (MinGW)

## ファイル構成

* `main.c` : シミュレーションのメインロジック、描画ループ、物理演算を含むソースコード

## 物理パラメータの仕様

プログラム内の構造体（`Ball`）で管理されているパラメータは以下の通りです。数値を調整することで挙動を自由に変更できます。

| パラメータ | 変数名 | 初期値 | 概要 |
| :--- | :--- | :--- | :--- |
| **X軸速度** | `ball.velocityX` | `2.0` | 1フレームあたりに右へ進むピクセル数 |
| **Y軸初速** | `ball.velocityY` | `-10.0` | バウンド時に上方向へ飛び出す初速（負の値で上へ） |
| **重力加速度** | `ball.accelY` | `0.3` | 1フレームごとにY軸速度に加算される落下加速度 |
| **サイズ** | `ball.size` | `10` | 描画される四角形の一辺のピクセルサイズ |

## ビルド・実行方法

1.  お使いの開発環境（Visual Studio等）に **DXライブラリ** を導入します。
2.  プロジェクトの構成プロパティで、DXライブラリの `include` パスおよび `lib` パスを正しく通します。
3.  `main.c` をプロジェクトに追加し、ビルド（コンパイル）を実行します。
4.  実行後、ウィンドウが立ち上がり白い四角形が放物線を描いてバウンドします。**ESCキー** を押すと安全にプログラムが終了します。
